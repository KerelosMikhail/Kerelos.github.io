My Old Grimoire website

Develop the back-end of a book rating website. Creating a server with Express and MongoDB, allowing users to register, log in, and rate books. The project includes user authentication, book management, and a rating system.

Code Link

Project Overview

Creating a server with Express and connect it to a MongoDB database. This involves setting up the server structure and managing communication between the server and database.

Development for data models and implement CRUD operations (Create, Read, Update, Delete) for managing books and ratings.

Implement a secure authentication system for website users. The project also include image upload and optimization management, as well as the functionality to rate books and calculate the average score.

Used Green Code best practices to reduce the ecological footprint of the website. Also Mongoose to model MongoDB data, and following MVC architecture (Model-View-Controller) to structure your application.
